# Habit Tracking & Verification App (Prototype)

## Overview
This is a mobile prototype developed for the **NovaUCD Student Enterprise Competition**.

The app demonstrates a system for tracking and verifying real-world activities such as study, fitness, and time away from phones.

---

## Features

- Activity tracking (Study, Fitness, Unplugged)
- Start/Stop timer system
- Daily activity logging
- Simulated verification process
- Two-page interface:
  - Timer view
  - Stats view with date navigation

---

## Tech Stack

- React Native (Expo)
- JavaScript

---

## Purpose

This project is a **functional prototype** used to demonstrate:
- Core product concept
- User interaction flow
- Feasibility of verified habit tracking
